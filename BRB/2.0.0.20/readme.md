## Основні зміни:

* Додано функціонал поповнення полиць зі складського приміщення магазину в модулі "Контроль цінників". 
Працівник може вказати кількіть товару, яку варто викласти на полиці зі складу

## Сумістність:

* BRB_Sync v.1.57 і вище
